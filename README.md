# JOBIN

## Features
- Infinite scroll
- User can View Jobs Based on the applied filter 

## Tech Stack
- React
- React Router Dom
- Material UI

## Inital Setup

1. Install Node
2. Install Node Modules
    ```
    npm install
    ```
3. Start React Application
    ```
    npm start
    ```
4. Build React Application
    ```
    npm run build
    ```
